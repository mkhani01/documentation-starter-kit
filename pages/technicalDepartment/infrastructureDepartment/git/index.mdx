# Git و GitLab

تمام سورس کد های محصولات نادین سافت در بستر git و در نرم افزار gitlab نگهداری میشود.

## Git چیست؟

`Git` یک سیستم کنترل نسخه توزیع‌شده (Distributed Version Control System - DVCS) است که توسعه‌دهندگان نرم‌افزار را قادر می‌سازد تا بر روی پروژه‌های نرم‌افزاری به صورت مشترک کار کنند. این سیستم امکان مدیریت تغییرات در کدهای نرم‌افزاری، همکاری مؤثر بین تیم‌ها و بازگرداندن پروژه‌ها به وضعیت‌های قبلی را فراهم می‌آورد.

### ویژگی‌های کلیدی Git:

- **توزیع‌شده**: هر توسعه‌دهنده یک کپی کامل از مخزن به همراه تاریخچه کامل تغییرات را بر روی سیستم خود دارد.
- **سرعت**: Git برای اجرای عملیات روی مخزن محلی طراحی شده است، بنابراین عملیات‌ها سریع‌تر هستند.
- **امنیت**: Git از رمزنگاری SHA-1 برای شناسایی تغییرات استفاده می‌کند، این امر اطمینان از تمامیت داده‌ها را فراهم می‌آورد.
- **انعطاف‌پذیری**: پشتیبانی از برنچ‌های چندگانه به توسعه‌دهندگان اجازه می‌دهد تا به صورت همزمان روی ویژگی‌ها یا نسخه‌های مختلف کار کنند.

## مزایای GitLab

`GitLab` یک پلتفرم گیت مبتنی بر وب است که فراتر از مدیریت مخازن گیت، امکاناتی چون CI/CD، مدیریت ایشوها، بررسی کدها، ویکی‌ها و بیشتر را در اختیار قرار می‌دهد.

### ویژگی‌ها و مزایای کلیدی GitLab:

- **یکپارچه‌سازی CI/CD**: امکان خودکارسازی آزمایش، ساخت، و انتشار نرم‌افزارها.
- **مدیریت پروژه**: ابزارهایی برای برنامه‌ریزی، پیگیری و اجرای پروژه‌های نرم‌افزاری.
- **دسترسی و کنترل امنیتی**: تنظیم سطوح دسترسی و حفاظت از کدها و داده‌ها.
- **مستندسازی و ویکی**: ایجاد و نگهداری مستندات پروژه در کنار کدها.
- **مرور و بررسی کد**: امکان ارسال درخواست‌های ادغام (Merge Requests) و بررسی کد توسط تیم.
- **پشتیبانی از DevOps**: ادغام تمام فازهای توسعه نرم‌افزار، از برنامه‌ریزی تا انتشار، در یک پلتفرم واحد.
- **پشتیبانی از کانتینرها و Kubernetes**: ادغام آسان با Docker و Kubernetes برای مدیریت کانتینرها و دپلویمنت‌ها.
- **انجمن و پشتیبانی**: دسترسی به یک انجمن فعال و منابع گسترده برای کمک به حل مشکلات و بهبود پروژه‌ها.

GitLab با ارائه این مجموعه از ویژگی‌ها و مزایا، یک راهکار جامع برای مدیریت دوره حیات توسعه نرم‌افزار (SDLC) ارائه می‌دهد، که باعث بهبود بهره‌وری تیم‌های توسعه و کیفیت نهایی محصولات نرم‌افزاری می‌شود.
